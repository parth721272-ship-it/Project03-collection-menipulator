# 🎓 Student Data Organizer (Python)

## 📌 Description

This is a simple **Student Data Organizer** program written in Python.
It allows users to add, display, update, and delete student records easily using a menu-driven interface.

---

## 🚀 Features

* ➕ Add new student
* 📋 Display all students
* ✏️ Update student information
* ❌ Delete student record
* 📚 Display all subjects offered
* 🔁 Easy-to-use menu system

---

## 🧾 Data Structures Used

* **List (****`students`****)** → Stores all student records
* **Dictionary (****`student_data`****)** → Stores details of each student
* **Tuple (****`id_dob`****)** → Combines Student ID and Date of Birth
* **Set (****`subjects`****)** → Stores unique subjects (no duplicates)

---

## ▶️ How to Run

1. Make sure Python is installed
2. Save the file as `student_data_organizer.py`
3. Run the program:

```bash
python student_data_organizer.py
```
